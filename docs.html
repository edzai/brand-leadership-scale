Endpoints

POST /api/v1/score

Submit 12 raw Likert-scale responses (1–7) and receive the computed Brand Leadership Score with tier classification and benchmark comparisons.

Request body:

{
  "brand": "Nike",
  "category": "Sports & Athletics",
  "scores": [6, 5, 6, 4, 5, 4, 6, 7, 6, 7, 7, 6]
}
FieldTypeRequiredDescription
brandstringYesBrand name being rated
categorystringNoBrand category (for category benchmarks)
scoresnumber[]Yes12 integers (1–7). Items 1–3: Quality, 4–6: Value, 7–9: Innovativeness, 10–12: Popularity

Response:

{
  "brand": "Nike",
  "overall": 5.75,
  "tier": "Brand Leader",
  "dimensions": {
    "quality": 5.67,
    "value": 4.33,
    "innovativeness": 6.33,
    "popularity": 6.67
  },
  "benchmarks": {
    "global": { "count": 1247, "overall": 4.21, ... },
    "brand":  { "count": 42, "overall": 5.58, ... },
    "category": { "count": 189, "overall": 4.85, ... }
  },
  "usage": { "month": "2026-04", "count": 15, "limit": 100 }
}

POST /api/v1/keys

Generate a free API key. One key per email address.

curl -X POST /api/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com"}'

GET /api/v1/keys/{key}

Check your API key status and current month usage.

Quick Start

// 1. Get a free API key
const res = await fetch('/api/v1/keys', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ email: 'you@example.com' })
});
const { api_key } = await res.json();

// 2. Score a brand
const result = await fetch('/api/v1/score', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': api_key
  },
  body: JSON.stringify({
    brand: 'Nike',
    category: 'Sports & Athletics',
    scores: [6, 5, 6, 4, 5, 4, 6, 7, 6, 7, 7, 6]
  })
});
const data = await result.json();
console.log(data.tier);       // "Brand Leader"
console.log(data.overall);    // 5.75
console.log(data.benchmarks); // { global: {...}, brand: {...} }
